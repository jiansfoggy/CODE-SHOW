### Numpy and Matplotlib

> Install on Ubuntu  

``sudo apt-get install python-numpy python-matplotlib python-pandas``  

> Install via pip  

``get-pip.py``  
``pip install numpy``  
``pip install matplotlib``  
``pip install pandas``  

> Install via Homebrew  

``brew install pkg-config``  
``pip install matplotlib``  
``pip install numpy``  
``pip install pandas``
